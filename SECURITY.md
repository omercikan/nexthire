# Security Policy

## Reporting a Vulnerability

If you discover any security vulnerabilities or issues in the NextHire project, please report them responsibly by contacting me directly:

- Email: nexthire00@gmail.com  
- LinkedIn: [https://www.linkedin.com/in/omercikan/](https://www.linkedin.com/in/omercikan/)

You can expect a response within a few business days after your report.

**Please do NOT open security-related issues publicly on GitHub to avoid exposing potential vulnerabilities before they are fixed.**
